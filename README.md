# NEON OVERDRIVE

横スクロールシューティングゲーム「NEON OVERDRIVE」へようこそ。
迫りくる敵を撃破し、巨大戦艦とのボスバトルに挑め！

## 概要
「NEON OVERDRIVE」は、ブラウザで動作するHTML5シューティングゲームです。
インストール不要で、ファイルをブラウザで開くだけですぐに遊べます。
「AUTO PILOT」システムを搭載しており、AIによる自動戦闘も可能です。

## インストール方法 / 起動方法
本ゲームはウェブブラウザ上で動作します。特別なインストーラーはありません。

1. フォルダ内の `index.html` をダブルクリック、またはブラウザ（Chrome, Firefox, Edgeなど）にドラッグ＆ドロップしてください。
2. ゲームが起動し、タイトル画面が表示されます。

## 操作方法

### PCでの操作
*   **移動**: 矢印キー (↑ ↓ ← →) または WASDキー
*   **攻撃**: スペースキー (または自動発射)
*   **自動操縦 (AUTO PILOT)**: `P` キー、または画面上の「AUTO PILOT」ボタンをクリック

### スマートフォン / タブレットでの操作
*   **移動**: 画面をタッチしてドラッグ
*   **攻撃**: 自動発射

## ゲームシステム解説
*   **武器レベルアップ**: "P" アイテムを取るとショットが強化されます。
*   **回復**: "H" アイテムで体力が回復します。
*   **ファンネル**: "F" アイテムで自機を追従する攻撃ユニット「ファンネル」を装備します。
*   **メガビーム**: "M" アイテムで強力なビームを発射します。
*   **設定 (SETTINGS)**: タイトル画面の「SETTINGS」から、自機や敵の画像（スキン）、BGMなどのURLを変更してカスタマイズ可能です。
