# NEON OVERDRIVE

**全包囲スクロールシューティングゲーム**
**INFINITE BATTLE SYSTEM**

迫りくる敵を撃破し、進化する巨大戦艦との無限バトルに挑め！
戦況に合わせて重力（スクロール方向）が変化する「GRAVITY SHIFT」システム搭載。

## 概要
「NEON OVERDRIVE」は、ブラウザで動作するHTML5シューティングゲームです。
インストール不要で、ファイルをブラウザで開くだけですぐに遊べます。
高度なAIによる自動戦闘システム「AUTO PILOT」を搭載し、放置プレイや観戦モードとしても楽しめます。

## 特徴
*   **全包囲スクロール**: "GRAVITY SHIFT" により、スクロール方向が上下左右、あるいはカオスに変化。あらゆる方向からの脅威に対応せよ。
*   **AUTO PILOT**: AIが自機を操作する自動操縦モード。
*   **RAMPAGE SYSTEM**: オートパイロット中に危機に陥ると「暴走モード」が発動。HP回復とともに無敵化し、質量を持った残像と共に敵を殲滅する。
*   **進化するボス**: 撃破するたびに強化される巨大戦艦。シールド、極太ビーム、そして触手による猛攻が待ち受ける。

## 操作方法

### PCでの操作
*   **移動**: 矢印キー (↑ ↓ ← →) または WASDキー
*   **攻撃**: スペースキー (押しっぱなしで連射)
*   **自動操縦 (AUTO PILOT)**: `P` キー、または画面上の「AUTO PILOT」ボタンをクリック
*   **設定**: タイトル画面の「SETTINGS」ボタン

### スマートフォン / タブレットでの操作
*   **移動**: 画面をタッチしてドラッグ
*   **攻撃**: 自動発射
*   **自動操縦**: 画面上の「AUTO PILOT」ボタンをタップ

## アイテムシステム
敵を倒すと出現するパワーアップアイテムを活用せよ。

*   **P (Power)**: ショットレベルアップ。拡散弾や攻撃力強化。
*   **H (Heal)**: 機体修復（HP回復）。
*   **F (Funnel)**: 自律攻撃ユニット「ファンネル」を装備。敵を自動追尾して攻撃。
*   **M (Mega Beam)**: 前方の敵を薙ぎ払う強力なメガビームを発射。
*   **S (Speed)**: 弾速強化 (Bullet Speed)。着弾までの時間を短縮。
*   **R (Rapid)**: 連射速度強化 (Rapid Fire)。弾幕密度を向上。
*   **A (Agility)**: 機動力強化 (Move Speed)。移動スピードアップ。

## 設定 (SETTINGS)
タイトル画面の「SETTINGS」から、ゲーム内の画像やBGMをカスタマイズできます。
画像URLやローカルファイルを指定することで、自機や敵、背景を好きな画像に変更可能です。
設定はブラウザに保存されます。
